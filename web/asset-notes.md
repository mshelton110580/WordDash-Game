# Tile Asset Notes

## Assets Available
- tile_normal.png: 256x256 wood Scrabble-style tile (warm wood grain, rounded corners)
- tile_selected.png: 256x256 brighter/lighter version of normal tile (selected state)
- tile_special_bomb.png: 256x256 wood tile with bomb icon overlay
- tile_special_cross.png: 256x256 wood tile with cross laser icon
- tile_special_laser.png: 256x256 wood tile with laser icon
- tile_special_mine.png: 256x256 wood tile with mine icon
- tile_special_wildcard.png: 256x256 wood tile with wildcard icon
- tile_ice_1.png: 256x256 ice-encased wood tile (2-hit state, more ice)
- tile_ice_2.png: 256x256 cracked ice tile (1-hit state, less ice)
- bg_game.png: 1170x2532 game background
- bg_menu.png: 1170x2532 menu background
- icon_bomb/cross/hint/laser/mine/shuffle: 128x128 power-up icons

## Design Theme
- Warm wood/Scrabble aesthetic
- Particle colors should be: warm dust/chips for normal, orange for bomb, blue for laser, purple for cross, yellow for wildcard, orange fragments for mine
